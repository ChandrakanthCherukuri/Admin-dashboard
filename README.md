# React Admin Dashboard

This is a fully customizable Admin Dashboard built with **React** and **Tailwind CSS**, featuring a sidebar, navbar, and dynamic pages including Dashboard, Calendar, Kanban, Charts, and Tables.

## 🚀 Features

- 📊 Dashboard Overview
- 📅 Calendar (via Syncfusion)
- 📋 Kanban Board
- 📈 Charts (Recharts)
- 📑 Data Tables
- 🌙 Light/Dark Theme Toggle
- ⚡ Fast and responsive design

## 🛠️ Tech Stack

- React `18.2.0`
- Tailwind CSS
- Syncfusion Components (Calendar, Kanban)
- Recharts
- React Router DOM
- React Icons
- Lucide React

---

## 🔧 Installation and Setup

### Prerequisites

Make sure **Node.js (v16 or higher)** and **npm** are installed on your system.

### Steps to run locally:

```bash
# Clone the repository
git clone https://github.com/yourusername/admin-dashboard.git
cd admin-dashboard

# Install dependencies
npm install

# Start the development server
npm start
